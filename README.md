# Build a base Microfrontend apps

Features:

- 🛠️ React + Vite: Host
- 🧰 Angular + Webpack: Remote
- 🪄 Vue3 + Vite: Remote
- 🎨 Module Federation with vite, webpack
- ↩️ UI: Shadcn with React, Vue, Angular (Spartan UI)
- ⌨️ State Management:
  - Client: Zustand
  - Server: Tanstack Query

### Install packages

```shell
pnpm i
```

### Setup .env file

Change .env.example => .env

### Start the app

```shell
pnpm dev
or
pnpm start
```
